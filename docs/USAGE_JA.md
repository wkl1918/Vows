## クイックスタート

1. バックエンド起動:
```powershell
conda activate VoxFlow
cd backend
python app.py
```

2. タスクスクリプト実行:
```powershell
cd ..
python run_task.py "D:/path/to/input.mp4" zh
```
または
```powershell
python run_task_with_subtitles.py "D:/path/to/input.mp4" zh
```

3. ドラッグ＆ドロップスクリプト:
- `一键处理视频.bat`
- `一键处理视频并加硬字幕.bat`
